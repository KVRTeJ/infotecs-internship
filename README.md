# infotecs-internship

## Description

Test task for the internship at infotecs. Here you can see multithreading, sockets and cmake.

## Technologies

- **Language:** C++17
- **Compiler:** GNU GCC
- **Build System:** CMake
- **OS:** Linux (development on macOS 13.6)
